Списки

Нумерованный список:

<dl> ― Definition List;
<dt> ― Definition Term;
<dd> ― Description Details;
Нумерованный список
― предназначен для перечисления элементов в точном порядке.
Маркированный список
― предназначен для перечисления элементов в произвольном порядке.
Список определений
― предназначен для перечисления терминов и разъяснений этих терминов в виде определений.
